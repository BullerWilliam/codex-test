# AI Console Chat

This folder contains a free anonymous cloud chat script you can use directly from the terminal.

## Run it

Double-click `run-chat.bat`

or run:

```powershell
& "C:\Users\tulli\Documents\GitHub\codex-test\chrome ai\run-chat.bat"
```

## What to expect

- A terminal chat starts immediately.
- Type normally to chat.
- Use `/reset` to clear conversation history.
- Use `/exit` to quit.

## Notes

- The script only uses files inside this `chrome ai` folder.
- It currently uses `https://text.pollinations.ai/openai`.
- No login or API key is required.
